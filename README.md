# NV Tasty Foods AI Assistant

AI-powered food ordering and order tracking assistant built using:

- n8n
- Google Gemini
- Google Sheets

## Features

### Menu Browsing
Users can view available food items.

### Order Placement
Users can place food orders with:

- Customer Name
- Phone Number
- Delivery Address
- Quantity

### Order Tracking
Users can track orders using phone number.

Example:

🍽️ Order Found

Order ID: ORD-20260621223844

Customer: Sunil

Food Item: Butter Chicken

Quantity: 3

📍 Delivery Address:
Nehru Nagar, Guntur

📦 Status: Order Placed

### FAQ Support
Answers restaurant questions using Google Sheets.

## Files

- nv-tasty-foods-workflow.json → n8n workflow export
- SYSTEM_PROMPT.md → AI Agent instructions
